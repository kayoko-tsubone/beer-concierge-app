# AIビールコンシェルジュ - Streamlit版

## 概要

その日の気分や天気、食べるものによって「今日はどんなビールを飲もうかな」と迷うことがよくあると思います。
このアプリは、そういった日常のちょっとした悩みを解決するために、機械学習を使っておすすめのビールを提案するものです。

ユーザーが「天気・気分・おつまみ」を選ぶだけで、それに合ったビールタイプを表示するシンプルなWebアプリとして作成しました。

## 課題

普段ビールを選ぶときに、なんとなくで選んでしまうことが多く、
「この料理にはどのビールが合うんだろう？」と考えることがありました。

そこで、条件に応じておすすめを提案してくれる仕組みがあれば便利だと思い、このアプリを作成しました。

## 実行環境

- **Python**: 3.11.0rc1以上
- **使用ライブラリ**:
  - `streamlit==1.56.0`: Webアプリケーションフレームワーク
  - `scikit-learn==1.8.0`: 機械学習モデル
  - `pandas==2.2.0`: データ処理
  - `joblib==1.4.2`: モデルの保存と読み込み

## 機械学習モデル

### 使用手法

**決定木（Decision Tree）**を採用しました。理由は以下の通りです：

- 処理の流れが比較的わかりやすい
- 少量のデータでも高い精度を発揮できる
- 初心者でも扱いやすい

### データセット

今回は自分で簡単なデータセットを作成しました。
入力データは以下の３つです。

- **天気**: 0:晴れ（暑い！）、1:くもり、2:雨
- **気分**: 0:リフレッシュしたい、1:リラックスしたい、2:ガッツリ飲みたい
- **おつまみ**: 0:揚げ物（唐揚げなど）、1:ナッツ・チーズ、2:スイーツ

### ビールの種類

- **0: IPA（インディア・ペールエール）**: 苦味と香りがガツンとくる、飲みごたえ抜群の一杯
- **1: HAZY IPA（ヘイジーIPA）**: 濁りがあってフルーティー、ジュースのような飲みやすさ
- **2: ペールエール**: 香り豊かでバランスの良い、クラフトビールの王道
- **3: ピルスナー**: 喉越し最高！キンキンに冷やして飲みたい定番ビール
- **4: スタウト（黒ビール）**: 濃厚でコクがある、スイーツとも相性抜群

### 学習結果

学習データに対する制度は100％となりました。

## 実行方法

### 1. ライブラリのインストール

```bash
pip install -r requirements.txt
```

### 2. モデルの学習（初回のみ）

```bash
python train_beer_model.py
```

このコマンドで、`beer_model.pkl`と`beer_labels.pkl`が生成されます。

### 3. Streamlitアプリの起動

```bash
streamlit run beer_app.py
```

ブラウザで `http://localhost:8501` にアクセスすると、アプリが表示されます。

## 使い方

1. **サイドバーで情報を入力**: 天気、気分、おつまみを選択
2. **「🎯 最高の一杯を探す！」ボタンをクリック**: AIが判定を開始
3. **結果を確認**: おすすめのビール、説明、確信度が表示されます

## 考察・反省

実際に自分で使ってみると、簡単な入力でもそれっぽい結果が出るのが面白いと感じました。
一方で、今回は自分で作成したシンプルなデータセットを使用しているため、現実の好みや個人差までは反映できていないと感じました。
また、機械学習のモデル自体はシンプルなものですが、アプリとして動かすことで実際に使える形にできた点はいい経験になりました。

## Renderでのデプロイ方法

### 1. GitHubにリポジトリを作成

```bash
git init
git add .
git commit -m "Initial commit: AIビア・コンシェルジュ"
git branch -M main
git remote add origin https://github.com/your-username/beer-concierge-app.git
git push -u origin main
```

### 2. Renderでのデプロイ

1. [Render](https://render.com)にログイン
2. 「New +」→「Web Service」を選択
3. GitHubリポジトリを接続
4. 以下の設定を行う：
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `streamlit run beer_app.py --server.port=$PORT --server.address=0.0.0.0`
5. 「Create Web Service」をクリック

デプロイが完了すると、永久的なURLが発行されます。
