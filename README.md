# AIビールコンシェルジュ - Streamlit版

## 概要

このプロジェクトは、機械学習を用いてユーザーの「今の状況（天気、気分、おつまみ）」に最適なビールを提案するStreamlitアプリケーションです。

## 解決する問題

「今日はどんなビールを飲もうかな？」「このおつまみにはどのビールが合うんだろう？」といった、ビール好きなら誰もが一度は抱える小さな悩みを解決します。

## 実行環境

- **Python**: 3.11.0rc1以上
- **主要ライブラリ**:
  - `streamlit==1.56.0`: Webアプリケーションフレームワーク
  - `scikit-learn==1.8.0`: 機械学習モデル
  - `pandas==2.2.0`: データ処理
  - `joblib==1.4.2`: モデルの保存と読み込み

## 機械学習モデル

### 使用手法

**決定木（Decision Tree）**を採用しました。理由は以下の通りです：

- 判断ロジックが比較的理解しやすい
- 少量のデータでも高い精度を発揮できる
- 初心者向けの成果物として適切

### データセット

オリジナルのビール相性データセットを作成しました。以下の3つの入力特徴量と、それに対応する最適なビールタイプで構成されています：

- **天気**: 0:晴れ（暑い！）、1:くもり、2:雨
- **気分**: 0:リフレッシュしたい、1:リラックスしたい、2:ガッツリ飲みたい
- **おつまみ**: 0:揚げ物（唐揚げなど）、1:ナッツ・チーズ、2:スイーツ

### ビールタイプ

- **0: IPA（インディア・ペールエール）**: 苦味と香りがガツンとくる、飲みごたえ抜群の一杯
- **1: HAZY IPA（ヘイジーIPA）**: 濁りがあってフルーティー、ジュースのような飲みやすさ
- **2: ペールエール**: 香り豊かでバランスの良い、クラフトビールの王道
- **3: ピルスナー**: 喉越し最高！キンキンに冷やして飲みたい定番ビール
- **4: スタウト（黒ビール）**: 濃厚でコクがある、スイーツとも相性抜群

### 学習結果

- **学習データの精度**: 100%

## ローカルでの実行方法

### 1. 依存ライブラリのインストール

```bash
pip install -r requirements.txt
```

### 2. モデルの学習（初回のみ）

```bash
python train_beer_model.py
```

このコマンドで、`beer_model.pkl`と`beer_labels.pkl`が生成されます。

### 3. Streamlitアプリの起動

```bash
streamlit run beer_app.py
```

ブラウザで `http://localhost:8501` にアクセスすると、アプリが表示されます。

## 使い方

1. **サイドバーで情報を入力**: 天気、気分、おつまみを選択
2. **「🎯 最高の一杯を探す！」ボタンをクリック**: AIが判定を開始
3. **結果を確認**: おすすめのビール、説明、確信度が表示されます

## 考察・反省

### 成果物の評価基準に沿った自己評価

- **アプリ動作**: 問題なくアプリは動作し、ユーザーが選択した状況に基づいて最適なビールを提案します。直感的なUIにより、シンプルで使いやすいアプリとなりました。
- **構成**: 目次を設け、適度な改行と見出しを用いて読みやすい構成を心がけました。考察・反省のセクションも設けています。
- **再現性**: 実行環境の情報、機械学習の手法（決定木）、オリジナルのデータセットの定義、モデルの学習・保存・読み込み、アプリの実装手順を明記しており、本READMEと提供されたコードがあれば再現可能です。

### 今後の改善点

- **データセットの拡充**: より多くのビール愛好家からのデータや、ビールの詳細な特徴（IBU、SRM、アルコール度数など）を取り入れることで、提案の幅と精度を向上させることができると思いました。
- **レコメンデーションの多様化**: 現在は単一のビールを提案していますが、複数の候補を提示したり、「なぜこのビールがおすすめなのか」という理由をAIが説明する機能を追加することで、ユーザー体験をさらに豊かにできます。
- **ユーザーフィードバック機能**: ユーザーが提案されたビールを飲んだ後の感想をフィードバックできる機能を実装し、そのデータをモデルの再学習に利用することで、AIがより賢くなる仕組みを構築できます。

## Renderでのデプロイ方法

### 1. GitHubにリポジトリを作成

```bash
git init
git add .
git commit -m "Initial commit: AIビア・コンシェルジュ"
git branch -M main
git remote add origin https://github.com/your-username/beer-concierge-app.git
git push -u origin main
```

### 2. Renderでのデプロイ

1. [Render](https://render.com)にログイン
2. 「New +」→「Web Service」を選択
3. GitHubリポジトリを接続
4. 以下の設定を行う：
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `streamlit run beer_app.py --server.port=$PORT --server.address=0.0.0.0`
5. 「Create Web Service」をクリック

デプロイが完了すると、永久的なURLが発行されます。
